# AI Incident Manager Frontend

![AI Incident Manager assistant](./public/Robot.png)

Frontend-only showcase for the AI Incident Manager portfolio demo.

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.

The showcase uses seeded incident data and simulated Slack/SMS notifications, so no backend or API keys are required.
